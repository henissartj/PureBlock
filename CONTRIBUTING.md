# Comment contribuer à PureBlock

Merci pour votre intérêt ! Voici comment participer :

## Étapes
1. **Fork** le projet
2. Créez une branche : `git checkout -b feat/ma-fonction`
3. Commitez clairement : `git commit -m "feat: ajoute le mode sombre"`
4. Poussez : `git push origin feat/ma-fonction`
5. Ouvrez une **Pull Request**

## Règles
- Respectez le style de code (ESLint si possible)
- Ajoutez des commentaires si nécessaire
- Testez sur Chrome **et** Firefox

[Ouvrir une Issue](https://github.com/henissartj/PureBlock/issues/new)
